Проект направлен на изучение языка Python, реальную практику разработки и работы с локальными сетями, а также ИИ.

UDP_Spam - написано на Python, спамит максимальными по весу UDP пакетами на адрес и порт, вписанный в начале
Производительность ~8-12 МБит
ICMP_Spam - написано на Python, спамит максимальными по весу Ping пакетами на адрес, вписанный в начале
Производительность - сколько позволяет сетевая карта
